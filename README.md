## Rust 관련 기초 명령어

```sh
curl https://sh.rustup.rs -sSf | sh

rustc hello.rs
rustc --version
rustup doc

cargo new --help
cargo new project_name --bin
cargo update
cargo check
cargo build
cargo run
```
